R: Representa a raiz de um documento HTML, é o contêiner para todos os outros elementos HTML (Exceto ). IMP!: Você deve sempre incluir o atributo LANG dentro da tag, para declarar o idioma da página web. O objetivo é ajudar os mecanismos de pesquisa e navegadores. R: É um contêiner para metadados (dados sobre dados) e é colocado entre a tag HTML e a tag BODY. Metadados são dados sobre o documento HTML. Os metadados não são exibidos. Os metadados normalmente definem o título do documento, conjunto de caracteres, estilos, scripts e outras metainformações. R: tag define o título do documento. O título deve ser somente texto e é mostrado na barra de título do navegador ou na aba da página. IMP!: É obrigatória em documentos HTML (não pode ter mais de uma dessa TAG no documento HTML) <body> R: Define o corpo do documento. IMP!: contém todo o conteúdo de um documento HTML, como títulos, parágrafos, imagens, hiperlinks, tabelas, listas, etc. <h1> ao <h6> R: São usadas para definir títulos HTML. IMP!: H1 define o mais importante, H6 menos importante. Use apenas um H1 por página - deve representar o título/assunto principal de toda a página. Além disso, não pule os níveis de título - comece com H1, depois use H2 e assim por diante. <p> R: Define um parágrafo. <br> R: Insere uma quebra de linha onde desejar.
